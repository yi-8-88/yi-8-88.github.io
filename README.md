# yi-8-88.github.io

Minimal dark personal site.

To add photos:
- Upload images to `assets/images/` (use Add file → Upload files on GitHub).
- Add entries in `_data/photos.yml` like:
  - title: "My photo"
    url: "/assets/images/photo1.jpg"
    thumb: "/assets/images/photo1.jpg"

To change social links:
- Edit `_config.yml` social entries.

Enable Pages:
- Repository → Settings → Pages → Branch: main, Folder: / (root).
